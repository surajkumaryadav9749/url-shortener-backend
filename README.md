A scalable URL Shortener backend built using **Node.js, Express, MongoDB, and Mongoose**.  
This project allows users to generate short URLs, redirect to original URLs, and track analytics such as visit history.

## Features

- Generate short URLs using nanoid
- Redirect short URLs to original URLs
- Track visit history (click analytics)
- RESTful API design
- Clean MVC-style project structure
- Ready for future frontend (React) integration
